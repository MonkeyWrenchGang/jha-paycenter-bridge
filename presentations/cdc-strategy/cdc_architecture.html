Phase 1

Native CDC + Debezium

Use SQL Server's built-in CDC feature with Debezium/Kafka. Zero licensing cost, operational overhead from SQL Agent and Kafka cluster. Best for fast start.

Phase 2 ⭐

Enterprise Log-Based CDC

IIDR or Striim read the transaction log directly. ~0% additional load on SQL Server. AG-aware, automated schema evolution. Striim recommended for native BigQuery integration.
